Description

BSPC is a tool designed to do the following:

  • Convert .BSP or .MAP files to .AAS files that the Gladiator bot needs to play a map.
  • Convert .BSP files to .MAP files. Handy if you lost your original .MAP file.
Note that it can read to be converted .BSP files directly out of .PAK or .SIN files.

BSPC can be used with all BSPs (game maps, mission pack maps, custom maps etc.) of the following games:

  • Quake 1
  • Quake 2
  • Heretic 2
  • Sin
  • Half-Life
System requirements

A PC running Quake2.
 

Usage

bspc [-<switch> [-<switch> ...]]

Example 1: bspc -bsp2aas znt2dm?.bsp -output e:\quake2\gladiator
Example 2: bspc -bsp2aas pak1.pak\maps/q2dm*.bsp -output e:\quake2\gladiator

Switches:
 
map2bsp <[pakfilefilter/]filefilter> = convert MAP to BSP
map2aas <[pakfilefilter/]filefilter> = convert MAP to AAS
bsp2map <[pakfilefilter/]filefilter> = convert BSP to MAP
bsp2bsp <[pakfilefilter/]filefilter> = convert BSP to BSP
bsp2aas <[pakfilefilter/]filefilter> = convert BSP to AAS
output <output path> = set output path
noverbose = disable verbose output
threads = number of threads to use
breath = breath first bsp building
nobrushmerge = don't merge brushes
noliquids = don't write liquids to map
freetree = free the bsp tree
nocsg = disables brush chopping

.PAK or .SIN files are accessed as if they are normal folders. For instance use "e:\quake2\gladiator\pak1.pak\maps/q2dm1.bsp" to access the map q2dm1.bsp from the pak1.pak file.

Multiple files may be listed after the map2bsp, map2aas, bsp2map, bsp2bsp or bsp2aas switches.
